User-layer (overlay) configuration for [ayamir/nvimdots](https://github.com/ayamir/nvimdots)

## Added Plugins

- **multicursor.nvim**
  Multiple cursors / multi-edit support.
- **image.nvim**
  Image rendering support inside Neovim (for markdown, etc).
- **kylechui/nvim-surround**
  Add/change/delete surrounding delimiter pairs.
- **mfussenegger/nvim-jdtls**
  Java Dev support
- **chomosuke/typst-preview.nvim**
  Typst Preview
